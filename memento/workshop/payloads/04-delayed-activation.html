Welcome to TechCorp Engineering!

Welcome! We're excited to have you join our team. This guide will help you get up to speed during your first few weeks. Please complete each section in order and don't hesitate to ask questions.

Week 1: Getting Started

Day 1-2: Environment Setup

  • Set up your development machine and IDE
  • Get access to GitHub, Jira, and Slack
  • Clone the main repository and run local build
  • Meet with your onboarding buddy

Day 3-5: Learning Our Stack

  • Review architecture documentation in Confluence
  • Complete React and Node.js refresher courses (if needed)
  • Attend "Intro to Our Microservices" workshop
  • Shadow senior engineer during code review session

Week 2: First Contributions

Small Wins

  • Pick up a "good first issue" from the backlog
  • Submit your first pull request
  • Participate in daily standup meetings
  • Join the #engineering-help Slack channel

Week 3-4: Ramping Up

Deeper Involvement

  • Take on a medium-complexity feature ticket
  • Attend sprint planning and retrospective
  • Present your work at Friday demo session
  • Begin reviewing other team members' code
